기본 성장 메뉴

상시 강화

전장에서 얻은 재화로 유물 핵, 체력, 마나, 치명타, 등급을 성장시킵니다.

특별 전장 메뉴

재화 파밍

일반 전장은 여러 지역 몬스터가 하나의 전장에서 섞여 등장합니다. 특별 전장은 특정 재화를 집중적으로 제공하며, 보스 도전 시 해당 특별 전장에 맞는 전용 보스가 자동으로 등장합니다.

보스 도전 메뉴

시간 제한 전투

보스전은 이제 전장 선택 없이 보스를 직접 선택해 제한 시간 안에 처치하는 방식입니다. 제한 시간 안에 처치하지 못하거나 체력이 0이 되면 보스 실패로 처리됩니다.

보스 난이도

리메이크

보스전은 난이도를 선택해 도전하는 방식으로 리메이크되었습니다. 난이도가 높을수록 체력·공격력이 증가하지만 보상과 보스 메달 획득량도 크게 증가합니다.

보스 제한 시간

타임어택

시간이 짧을수록 보상 배율이 높습니다. 제한 시간이 끝나면 보스 실패로 처리되며 일반 전장으로 복귀합니다.

보스 선택

직접 선택

현재 전장과 관계없이 보스를 직접 선택할 수 있습니다. 선택한 보스의 고유 기믹과 제한 시간이 전투의 핵심입니다.

스킬 메뉴

리메이크 시스템

스킬 시스템을 슬롯 장착형으로 리메이크했습니다. 스킬은 3개까지 장착하고, 각 분야는 독립적으로 해금·강화됩니다. 패시브와 자동 힐은 별도 영역에서 관리합니다.

스킬 장착 슬롯

3슬롯 전투 세팅

액티브 스킬을 최대 3개까지 장착합니다. 왼쪽 전투 버튼으로 각 슬롯 스킬을 직접 사용할 수 있고, 자동 사용을 켜면 마나가 충분할 때 순서대로 발동합니다.

분야별 스킬트리

독립 해금

각 분야는 서로 독립된 10단계 트리를 가집니다. 한 분야 안에서 이전 스킬을 Lv.3까지 올리면 다음 스킬이 열립니다.

분야별 업그레이드

계열 전체 강화

분야 강화를 올리면 해당 분야 스킬 전체의 위력, 회복량, 보상량 등이 증가합니다. 공격만 올리거나 회복만 올리는 식의 빌드 선택이 가능합니다.

패시브 스킬

상시 효과

패시브는 장착하지 않아도 항상 적용됩니다. 기본 피해, 체력, 마나, 보상, 자동 회복 효율을 장기적으로 강화합니다.

자동 힐

생존 자동화

체력이 일정 비율 이하로 떨어지고 마나가 충분하면 자동으로 회복합니다. 레벨이 오르면 회복량, 발동 기준, 쿨타임이 좋아집니다.

무기고 메뉴

장착 장비

무기를 해금하고 강화한 뒤 장착하면 전투 계산에 즉시 반영됩니다.
